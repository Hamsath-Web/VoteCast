# **App Name**: VoteCast

## Core Features:

- Voting History: Displays a historical record of past votings, including details and outcomes.
- Voting Creation: Allows admin to create new votings, define contestants, and upload related media (face images, team logo).
- Contestant Management: The user enters contestant data. This includes contestant names, two face images, and one team logo for each contestant.
- Voting Interface: Presents all contestants to voters, enabling selection of one contestant until voting is completed.
- Results Display: Shows the results of the voting, highlighting the winner and providing a vote count for all contestants.

## Style Guidelines:

- Primary color: A calm, authoritative blue (#4682B4) to evoke trust and reliability.
- Background color: A light, desaturated blue (#E6E8EA) for a clean and professional feel.
- Accent color: An energizing yellow (#FFD700) to draw attention to key actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use simple, geometric icons to represent voting actions and results.
- Employ a clean, card-based layout to display voting options and results clearly.
- Subtle animations on vote submission and result updates to provide feedback and engagement.
